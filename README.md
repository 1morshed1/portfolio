# Md. Morshed Jamal — Witcher 3 Themed Portfolio

A personal portfolio website reimagined as a Witcher 3: Wild Hunt experience. Dark fantasy aesthetic with interactive game UI elements, built as a single static HTML page.

## Features

### Witcher 3 Aesthetic
- **Cinzel & EB Garamond** gothic serif typography
- Gold/amber color palette on dark leather-toned backgrounds
- Parchment textures, warm vignette, ornate corner decorations
- Custom crosshair cursor with golden pointer for interactive elements

### Interactive Elements
- **Loading Screen** — Wolf medallion pulse animation with random Witcher-themed tips
- **Ember Particle System** — Toggle between ember, snow, and fog weather effects
- **Map Waypoint Navigation** — Dot-and-line nav with active section ping animation
- **Inventory Tooltips** — Hover skills to see W3-style item tooltips with rarity colors (Legendary/Epic/Rare/Common) and flavor text
- **Click-to-Flip Cards** — All skill and project cards start face-down. Gwent cards show green backs with diamond pattern and faction emblem. Contract cards show yellowed parchment with wax seals. Click to flip and reveal content
- **Quest Completion Notifications** — Slide-in popup when scrolling to each section
- **Meditation Clock** — Rotating day/night cycle SVG in footer

### Section Designs
- **Bestiary About** — Structured as Class, Habitat, Strengths, Weakness, Trophies, Origin fields
- **Quest Log (Experience)** — Active Quest / Contract Completed badges, XP duration tags, sword bullet points
- **Gwent Board (Skills)** — Cards arranged in Melee/Ranged/Siege rows with faction colors (Nilfgaard, Monsters, Skellige, Scoia'tael, Northern Realms), strength badges, mutagen slots, and row score totals
- **Contract Board (Projects)** — Notice board with wax seals, difficulty stars, tilted parchment cards, "Take Contract" buttons
- **Witcher Signs** — Quen, Igni, Axii, Aard, Yrden SVG glyphs on each section title

### Other Details
- Wolf medallion SVG logo with hover glow
- Level 28 Witcher progress bar
- Death March difficulty badge
- Diamond SVG dividers between sections
- Gold scrollbar, selection highlight, progress bar
- Mobile-responsive with wolf medallion hamburger menu
- Section reveal animations with golden shimmer

## Tech Stack

- **HTML5** — Single-file static page
- **Tailwind CSS** — Via CDN for utility classes
- **Vanilla JavaScript** — Particles, typing effect, scroll observers, card flip
- **Google Fonts** — Cinzel, Cinzel Decorative, EB Garamond
- **SVG** — All icons, signs, medallions, and decorations are inline SVG

## Setup

```bash
git clone https://github.com/1morshed1/portfolio
cd portfolio
# Open index.html in any browser — no build step required
```
